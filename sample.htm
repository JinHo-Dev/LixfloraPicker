LixfloraPicker

This is a sample page for LixfloraPicker

https://github.com/JinHo-Dev/LixfloraPicker

What's this?

This is Apple-Style picker performs as multiple <select>
As you see, It's virtually rolled-up
You can drag, touch, wheel or just click to select things you want
It is made of pure javascript

How do I APPLY this?

Make a DIV object whose height is 160px and class is "LixfloraPicker"
Make your own SELECT objects inside that DIV
[Example] 

<div class="LixfloraPicker" style="height: 160px; width: 200px;"> 
    <select> 
        <option> 1 </option>
        <option selected> 2 </option>
        <option> 3 </option>
    </select> 
    <select infinite> 
        <option> A </option>
        <option> B </option>
        <option> C </option>
    </select> 
</div>

How can I get the values?

Just by getting the value of original <select>, you can get updated value

I want to update the lists of select option

Just modify options of original <select>
Then execute: [LixfloraPicker object's DOM].LixfloraPicker.reload([index of select]);

I want to make it scroll infinitely

Set attribute "infinite" to your <select> like <select infinite>

I want to preselect the option

Set attribute "selected" to your <option> like <option selected>

Theme

You can adjust "background-color" of Div style
You can adjust "color" of Div style

OnChange Event

If the value changes, it triggers onchange event to you select object
And you can modify your options in you selects
To reflect changes, execute function: [your LixfloraPicker object].LixfloraPicker.reloadAll();
